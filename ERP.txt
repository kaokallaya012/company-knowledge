Start System: ข้อมูลพื้นฐาน
* ประเภทเจ้าหนี้: คือกลุ่มคนที่เราไปซื้อของเขามาแล้วเป็นหนี้เขาอยู่ เพิ่มข้อมูลเจ้าหนี้ ลบและแก้ไขได้ โดยเพิ่มประเภทและรายละเอียด เมื่อเพิ่มเรียบร้อยแล้วให้กดปุ่ม Add ข้อมูลที่เพิ่มเข้าไปจะไปปรากฎอยู่ในตาราง


* ประเภทลูกหนี้: แสดงข้อมูลประเภทของลูกหนี้ ว่าเป็นลูกหนี้ประเภทใด
   * กลุ่มผู้ติดต่อ(1) เริ่มแรกต้องสร้างกลุ่มผู้ติดต่อก่อนเพื่อที่จะสามารถไปเพิ่มข้อมูลผู้ติดต่อได้ เราต้องรู้ก่อนว่าผู้ติดต่อเราคือใคร ทั้งยังสามารถลบ แก้ไข หรือดูรายการผู้ติดต่อได้ แต่กรณีการลบจะสามารถลบได้เฉพาะกลุ่มผู้ติดต่อที่ยังไม่มีข้อมูล การระบุข้อมูลควรใส่รหัสกลุ่มและรายละเอียดให้สอดคล้องกัน เริ่มเลขที่ คือจะเริ่มนับเลขที่เท่าไหร่หรือใส่0เป็นdefaultได้ จากนั้นเลือกประเภทว่าเป็นเจ้าหนี้หรือลูกหนี้ รหัสผังบัญชีจะเป็นในส่วนของบัญชีที่ต้องใส่หรือหากมีไว้อยู่แล้วสามารถกดดูและเลือกได้เลย เมื่อระบุเสร็จแล้วให้กด Add ข้อมูลที่เพิ่มเข้าไปจะปรากฎอยู่ที่ตารางด้านข้าง


   * กลุ่มลูกหนี้ กลุ่มบริษัทหรือบริษัทที่มีหลายสาขา ซึ่งมีภาวะหนี้สินรวมกัน เช่น ร้าน7/11


   * ประเภทธุรกิจ สามารถเพิ่ม ลบ แก้ไขได้ เพื่อให้ทราบว่าธุรกรรมนั้นๆเป็นประเภทอะไร ซึ่งจำแนกได้หลายประเภท


   * เกรดลูกค้า กรอกข้อมูลเครดิตที่ลูกค้าสามารถค้างชำระได้ตามเกรดที่แยกไว้ ตามยอดขายและจำนวนวันเครดิต


   * รายชื่อผู้ติดต่อ แสดงข้อมูลกลุ่มผู้ติดต่อที่เราเพิ่มไว้


* ข้อมูลการติดต่อ: กรอกข้อมูลตามช่องใส่ข้อมูลจากบนลงล่างโดยเน้นที่หัวข้อสีแดง กรณีเลขประจำตัวผู้เสียภาษี หากไม่มีไม่จำเป็นต้องใส่ หากใส่ไปแล้วต้องใส่ให้ถูกต้อง หลังจากกรอกข้อมูลเรียบร้อยแล้วให้กด Add จากนั้นข้อมูลจะเข้าสู่ระบบและสามารถดูข้อมูลลูกค้าและผู้ขายได้ ส่วนตรงราคาขาย สมมติเรามีลูกค้าหลายประเภททั้งซื้อเยอะหรือน้อย เราสามารถระบุราคาขายได้หากซื้อเยอะก็ราคาถูกลง หากซื้อน้อยก็ราคาตามปกติ


* สาขา/หน่วยงาน: เพิ่ม ลบ แก้ไขหน่วยงาน พนักงานและสาขาได้
   * หน่วยงาน ทำการเพิ่มหน่วยงานก่อนเพื่อที่จะสามารถเพิ่มพนักงานได้ สามารถแก้ไขได้ หน่วยงานไหนที่ยังไม่มีข้อมูลพนักงานจะสามารถลบได้ 
   * พนักงาน เพิ่มรายชื่อพนักงานและเลือกหน่วยงานที่เพิ่มไว้ก่อนหน้านี้
   * สาขา เพิ่มสาขาของบริษัทนั้นๆ ใช้งานสำหรับภาษีซื้อ-ขายเท่านั้น ไม่ใช้กับงบการเงิน


* ข้อมูลคลังสินค้า: เพิ่ม ลบ แก้ไขข้อมูลคลังสินค้า การเพิ่มข้อมูลคลังสินค้า ระบุข้อมูลลงไปให้เรียบร้อย


* ประเภทและกลุ่มสินค้า: สามารถเพิ่มรหัสกลุ่มสินค้าได้ ต้องทำการเพิ่มประเภทสินค้าก่อน


* ข้อมูลเขตการขาย - ทีมขาย: สามารถเพิ่ม ลบ แก้ไข เขตการขายได้ 


* ข้อมูลสินค้า: เพิ่มข้อมูลสินค้าโดยระบุกลุ่มของสินค้าให้ถูกต้อง


* ข้อมูลธนาคาร - สาขา: ทำการเพิ่มรหัสและชื่อธนาคาร จากนั้นทำการเพิ่มข้อมูลบัญชีธนาคาร เพื่อใช้ในการทำธุรกรรมต่างๆ


* ข้อมูลสกุลเงิน สามารถเพิ่มข้อมูลสกุลเงินต่างๆ ตามอัตราการแลกเปลี่ยน สำหรับกรณีการรับหรือจ่ายเงินที่ไม่ใช่เพียงสกุลเงินไทยเท่านั้น


* สมุดบัญชี: เพิ่มข้อมูลสมุดบัญชีได้ โดยจำเป็นต้องระบุรหัสสมุด อักษรนำ ชื่อสมุดและเลขที่ล่าสุด ซึ่งหากไม่ทราบสามารถกดปุ่มHelp เพื่อดูได้ว่าควรตั้งยังไง


* ผังบัญชี เพิ่มรหัสและชื่อบัญชีเพื่อใช้ในการทำธุรกรรม งบการเงินต่างๆ


* เชื่อมโยงบัญชี


* ข้อมูลอื่นๆ แสดงข้อมูลการคืนสินค้า สินค้าหมดอายุ สินค้าไม่ได้มาตรฐาน เหตุผลใบPO PR SO


* รายงาน รวบรวมและแสดงข้อมูลต่างๆที่เพิ่มเข้าไปตามหัวข้อ








Purchase System: ระบบซื้อ
* ใบขอซื้อ (PR) การสร้างใบขอซื้อ ระบุข้อมูลที่จำเป็นลงไปเพื่อสร้างหัวเอกสาร จากนั้นเลือกผู้อนุมัติที่ต้องอนุมัติให้เรา และกด Add เมื่อเพิ่มเสร็จแล้วจะมีช่องให้ใส่ข้อมูลสินค้า จำนวนและราคาที่ต้องการ จากนั้นกด Add เมื่อเพิ่มไปแล้วจะเห็นว่าสินค้าที่เราเพิ่มจะไปปรากฎอยู่ตรงกลางหน้า เมื่อเพิ่มข้อมูลสินค้าที่ต้องการขอซื้อจนครบแล้ว ให้เปลี่ยน PR Status จาก O/ระหว่างบันทึกข้อมูล เป็น W/รออนุมัติ และกด Update ในส่วนนี้เมื่อเปลี่ยนเป็นสเตตัส W แล้วจะไม่สามารถแก้ไข PR ได้อีกแต่ยังสามารถไปเพิ่มสินค้าที่ต้องการซื้อได้อีกที่หน้าใบสั่งซื้อ จากนั้นต้องรอให้ผู้อนุมัติมาอนุมัติให้ หากผู้อนุมัติเป็นคนละคนก็ต้องรอจนกว่าจะอนุมัติหรือไม่อนุมัติทั้งสองคน เมื่ออนุมัติแล้วใบขอซื้อจะส่งไปหาหน้าใบสั่งซื้อ เราสามารถ print PR


* ใบสั่งซื้อ (PO) การสร้างใบสั่งซื้อจะทำได้เมื่อสร้างใบขอซื้อและได้รับการอนุมัติแล้วหรือสร้างขึ้นใหม่ได้เลย ระบุข้อมูลลงไปให้เรียบร้อยหากมีใบขอซื้อให้ใส่ลงไปด้วยเพื่ออ้างอิง จากนั้นเลือกผู้อนุมัติและกด Add จากนั้นเพิ่มข้อมูลสินค้าที่จะสั่งซื้อลงไปแต่หากจะสั่งซื้อสินค้าโดยอ้างอิงจาก PR ให้เลือกส่วน Copy PR เมื่อคลิกจะแสดงข้อมูลสินค้าที่ส่งมาจากใบขอสินค้าสามารถเลือกได้ว่าจะสั่งสินค้าไหนบ้าง สามารถคลิกทีเดียวได้หากจะสั่งซื้อจากบริษัทเดียวกัน หรือหากคนละบริษัทก็สามารถเลือกแค่ที่มีก็ได้ เมื่อเลือกได้แล้วให้กด Add ระบบจะทำการคำนวณราคาสินค้าและภาษีให้ เมื่อสั่งสินค้าที่ต้องการครบแล้วให้เปลี่ยน PO Status จาก O/ระหว่างบันทึกข้อมูล เป็น W/รออนุมัติ และกด Update ในส่วนนี้เมื่อเปลี่ยนเป็นสเตตัส W แล้วจะไม่สามารถแก้ไข PO ได้อีก จากนั้นต้องรอให้ผู้อนุมัติมาอนุมัติให้ หากผู้อนุมัติเป็นคนละคนก็ต้องรอจนกว่าจะอนุมัติหรือไม่อนุมัติทั้งสองคน เมื่ออนุมัติแล้วใบสั่งซื้อจะส่งไปหาผู้ขายที่เราทำการสั่งซื้อ สามารถ print PO ในและต่างประเทศได้


* ซื้อสินค้า(ได้รับสินค้า) ในหน้านี้เราจำเป็นต้องรู้รหัสผู้ขาย หากไม่รู้เราจะหาข้อมูลสินค้าไม่เจอ เมื่อระบุข้อมูลเรียบร้อยแล้ว หากเป็นสินค้าที่สั่งซื้อไป ให้คลิก Copy PO จากนั้นเลือกรายการสินค้าที่ได้รับเข้าสต็อกและกด Add และสามารถเข้าไปเช็คสต็อกได้ สามารถ print ใบรับสินค้าได้


* ลดหนี้ซื้อ ลดราคา กรณีการลดหนี้เนื่องจากการลดราคา จำเป็นต้องรู้รหัสผู้ขายและรหัสใบซื้อสินค้าที่จะนำไปลดราคา จำนวนสินค้า


* ลดหนี้ซื้อ คืนสินค้า การคืนสินค้าให้ผู้ขายไม่ว่าจะกรณีใดก็ตาม และต้องมีการออกใบลดหนี้กรณีการลดหนี้เนื่องจากการคืนสินค้า จำเป็นต้องรู้รหัสผู้ขายและรหัสสินค้าที่ต้องการจะคืนสินค้า กรณีที่เก็บสินค้ามีหลายคลังจำเป็นต้องเลือกคลังสินค้าให้ถูกต้องด้วย หากเลือกคลังสินค้าผิด ระบบจะหาสินค้าไม่เจอและไม่สามารถคืนสินค้าได้
